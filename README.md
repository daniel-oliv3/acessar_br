Cria o projeto react
### npx create-react-app my_app

Executar o projeto
### npm start

Versão do node para instalar o sass
### node -v

Instalando o SASS
Tabela Node Sass - https://www.npmjs.com/package/node-sass
Versão-6
### npm install node-sass@6

Atualizar os pacotes de dependências
### npm update

Executar o projeto
### npm run start
